# Личный дневник на Django

Веб-приложение для ведения личного дневника с регистрацией, поиском и полным CRUD.

## Стек технологий
- Python 3.13
- Django 6.0
- PostgreSQL 15
- Bootstrap 5
- Docker / docker-compose
- GitHub Actions (CI/CD)

## Функционал
- Регистрация и вход (Django auth)
- Создание, редактирование, удаление записей (только своих)
- Просмотр списка и детальной страницы
- Поиск по заголовку и содержимому
- Адаптивный интерфейс на Bootstrap
- Админ-панель Django
